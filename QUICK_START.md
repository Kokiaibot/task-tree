## 📖 基本的な使い方

### 1. ツリービュー（構造画面）

#### タスクの作成
- **Enter**: 同じ階層に新しいタスクを追加
- **Ctrl + Enter**: 選択中のタスクの子タスクを追加
- **Shift + Enter**: 新しいプロジェクトを追加

#### タスクの編集
- タスク名をクリックして直接編集
- 日付フィールドをクリックして期限を設定
- ユーザーアイコンをクリックして担当者を変更

#### タスクの削除
- **Backspace**: 空欄時にタスクを削除
- **Ctrl + Delete**: 強制的にタスクを削除
- ゴミ箱アイコンをクリック

#### タスクの完了
- チェックボックスをクリック
- **Ctrl + Space**: キーボードショートカット

#### タスクタイプの切り替え
- **T/Iボタン**: タスク(Task)とイシュー(Issue)を切り替え
  - **Task**: 実行可能な具体的な作業（担当者・期限あり）
  - **Issue**: 課題や問題（抽象的、サブタスクの親として使用）

#### 詳細説明の追加
- 📄アイコンをクリックして説明エリアを表示
- メモや詳細情報を記入

### 2. 週間パイプラインビュー

- 左サイドバーの**カレンダーアイコン**をクリック
- 7日間のタスクをカラムで表示
- **ドラッグ&ドロップ**で期限を変更
- タスクカードをクリックして詳細を表示

### 3. ガントチャートビュー（タイムライン）

- 左サイドバーの**グラフアイコン**をクリック
- タスクのタイムラインを可視化
- 期間の長さを視覚的に確認
- 横スクロールで未来のタスクを確認
- **今日へ移動ボタン**: 現在の日付にスクロール

### 4. チーム管理

- 右上の**TEAMエリア**をクリック
- メンバーの追加・削除
- メンバーの名前と色を編集
- タスク割り当て時にメンバーを選択

## ⌨️ キーボードショートカット

### タスク作成
| キー | 動作 |
|------|------|
| Enter | 同じ階層に追加 |
| Ctrl + Enter | 子タスクを追加 |
| Shift + Enter | 新規プロジェクト |

### タスク削除
| キー | 動作 |
|------|------|
| Backspace | 空欄時に削除 |
| Ctrl + Delete | 強制削除 |

### タスク操作
| キー | 動作 |
|------|------|
| Ctrl + Space | 完了/未完了切替 |
| Ctrl + Shift | ビュー切替 |

## 💡 使用例

### プロジェクト管理
1. プロジェクト名を作成（Issue）
2. 主要なフェーズを子として追加（Issue）
3. 各フェーズの具体的タスクを追加（Task）
4. 担当者と期限を設定
5. 週間ビューで進捗確認
6. ガントチャートで全体スケジュール確認

### 個人タスク管理
1. カテゴリ別にプロジェクトを作成
2. タスクを細分化して追加
3. 期限を設定して週間ビューで管理
4. 完了したらチェック

### チーム協業
1. チームメンバーを追加
2. タスクに担当者を割り当て
3. 各メンバーの色で視覚的に識別
4. 週間ビューで負荷バランスを確認

## 💾 データの保存

- **自動保存**: 全ての変更は自動的にブラウザに保存されます
- **LocalStorage**: データはブラウザのLocalStorageに保存
- **注意**: 
  - ブラウザのキャッシュをクリアするとデータが消えます
  - 異なるデバイスやブラウザではデータは共有されません
  - 定期的にエクスポート機能（将来実装予定）でバックアップを推奨

## 🎨 カスタマイズ

### チームメンバーの色
- 7色から選択可能
- 各メンバーに一意の色を割り当て
- 視覚的にタスクの担当者を識別

### タスクの階層構造
- 無制限のネスト（階層）が可能
- 折りたたみ/展開で見やすく管理
- 親タスクの完了時に子タスクも自動完了

## 🔄 ビューの切り替え

左サイドバーのアイコンで瞬時に切り替え：
- **格子アイコン**: ツリービュー（詳細編集）
- **カレンダーアイコン**: 週間ビュー（日次管理）
- **グラフアイコン**: ガントチャート（スケジュール確認）

## ❓ ヘルプ

- 左サイドバー下部の**？アイコン**をクリック
- ショートカットキー一覧を表示
- 操作方法を確認

## 🐛 トラブルシューティング

### データが消えた
- ブラウザのキャッシュをクリアした可能性があります
- ブラウザの開発者ツール（F12）→ Application → Local Storage を確認

### 動作が重い
- ブラウザのタブを閉じて再起動
- 大量のタスク（1000+）がある場合は分割を推奨

### 表示が崩れる
- ブラウザのキャッシュをクリア（Ctrl + Shift + Delete）
- ページをリロード（F5）

## 📱 モバイル対応

- レスポンシブデザイン対応
- スマートフォンやタブレットでも使用可能
- タッチ操作でドラッグ&ドロップが可能
